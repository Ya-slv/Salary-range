# Руководство для ML-инженера (ml_readme.md)

В данном файле описана структура построения, предобработки данных и обучения ML-модели прогнозирования зарплатной вилки.

## 1. Задача и метод
Для прогнозирования зарплатного диапазона используется **одновременное предсказание двух квантилей** с помощью функции потерь CatBoost:
* **Квантиль 0.1** (нижняя граница вилки).
* **Квантиль 0.9** (верхняя граница вилки).

Это позволяет обойтись одной компактной и быстрой моделью `MultiQuantile` вместо обучения двух независимых регрессоров.

## 2. Данные и предобработка
* **Исходный датасет**: `new model/Z_new.csv` (разделитель `;`, кодировка `utf-8-sig`).
* **База рекомендаций**: `new model/data_for_recommendations.csv`.

### Процесс очистки данных (`train_fork_model.py`):
1. **Фильтрация нетехнических ролей**: Удаляются строки, содержащие нецелевые паттерны в названии вакансии (`продавец`, `консультант`, `станок`, `чпу`, `фрезеровщик`, `зубной`, `техник` и т.д.).
2. **Пропуски**: Удаляются строки без указания зарплаты (`salary`) или названия (`name`).
3. **Фильтрация по наличию навыков (Важно)**: Из обучения полностью исключены строки с пустыми навыками, пустыми списками или значениями `nan` (`['', 'nan', 'NaN', '[]']`).
4. **Трансформация навыков**: Строковые представления списков (например, `['Python', 'Git']`) переводятся в плоские текстовые строки разделенные запятой (`"Python, Git"`) для правильной работы текстового анализатора CatBoost.

## 3. Признаки (Features)
* **Категориальные признаки**: `city`, `experience`, `schedule`, `role`, `industry`, `employer`.
* **Текстовые признаки**: `name` (название вакансии), `skills` (список навыков).
* **Целевая переменная**: `salary`.

## 4. Обучение модели
* **Разбиение данных**: 93% обучающая выборка, 7% валидационная выборка (`test_size=0.07`) — для обучения на максимальном количестве данных.
* **Параметры CatBoostRegressor**:
  ```python
  CatBoostRegressor(
      iterations=1200,
      learning_rate=0.05,
      depth=6,
      cat_features=cat_features,
      text_features=text_features,
      loss_function='MultiQuantile:alpha=0.1,0.9',
      random_seed=42
  )
  ```

## 5. Выходные артефакты
После обучения скрипт создаёт:
1. `data/model.cbm` — бинарный файл обученной CatBoost модели.
2. `data/vacancies.csv` — очищенная база вакансий, используемая для сопоставления навыков кандидатов и выдачи персональных рекомендаций на бэкенде.
