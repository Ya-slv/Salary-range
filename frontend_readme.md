# Руководство для Frontend-разработчика (frontend_readme.md)

В данном файле описана структура фронтенда, особенности стилизации в стиле Т-Банка и логика интерактивных элементов.

## 1. Технологический стек
* **Фреймворк**: React 18
* **Инструмент сборки**: Vite
* **Стилизация**: Vanilla CSS (для максимальной производительности и гибкости)

## 2. Архитектура и компоненты (`App.jsx`)
Главный файл приложения реализует интерфейс калькулятора, разделённый на две основные зоны:
1. **Левая колонка (Форма ввода)**:
   * **Вкладка «Заполнить вручную»**: содержит текстовые инпуты, селекторы (город, опыт, тип занятости) и интерактивный блок добавления навыков.
   * **Вкладка «Загрузить резюме»**: область Drag-and-drop для загрузки файлов `.pdf`, `.docx`, `.doc`, `.txt` размером до 5 МБ. При загрузке отправляет запрос на `/api/parse_resume/` для автоматического заполнения полей формы.
2. **Правая колонка (Результаты расчёта)**:
   * Визуализация зарплатного диапазона (вилка `min_salary` – `max_salary`).
   * Индикатор положения вилки на шкале доходов.
   * Список рекомендаций — недостающих навыков, которые помогут увеличить доход.

## 3. Логика кнопки «Пересчитать»
Для улучшения пользовательского опыта (UX) кнопки расчёта работают динамически:
* Если расчёт ещё не производился, кнопка имеет текст `Рассчитать зарплату` (или `Распарсить резюме и рассчитать`).
* Если результаты уже на экране и пользователь внёс изменения, текст кнопки меняется на **`Пересчитать`** (или **`Пересчитать и обновить`**). Это подсказывает пользователю, что он может скорректировать данные и обновить прогноз в реальном времени.

## 4. Дизайн-система Т-Банка (`index.css`)
Интерфейс стилизован в фирменной желто-бело-черной гамме Т-Банка.

### Ключевые CSS-переменные (`:root`):
* `--primary`: `#FFDD2D` (Фирменный жёлтый цвет Т-Банка).
* `--bg-base`: `#0B0B0C` (Премиальный тёмный фон).
* `--bg-surface`: `#141416` (Цвет карточек и контейнеров).
* `--color-text`: `#FFFFFF` (Белый текст).
* `--color-text-muted`: `#8E8E93` (Серый приглушённый текст).

### Визуальные эффекты:
* **Кнопки**: Высококонтрастный жёлтый фон с чёрным текстом (`color: #111112`). Имеют микроанимации увеличения при наведении (`scale`) и мягкую пульсацию свечения (`animate-pulse-glow`).
* **Карточки**: При наведении на карточку появляется тонкая жёлтая граница и лёгкая тень, создающая объём.
* **Тэги навыков**: Окрашены в полупрозрачный жёлтый цвет с чёткой контрастной границей.
