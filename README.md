# Salary-range

Сервис, который анализирует резюме и выдает предсказание зарплатной вилки на основе машинного обучения, а также дает рекомендации по прокачке навыков для достижения более высокого дохода.

## Описание проекта
Приложение состоит из трех ключевых компонентов:
1. **Frontend (React + Vite)** — удобный и современный пользовательский интерфейс для ручного ввода данных.
2. **Backend (Django REST Framework)** — серверная часть, обрабатывающая запросы, парсинг резюме через локальную LLM и взаимодействие с ML-моделями.
3. **ML-модели (CatBoost + LLaMA)** — предсказание рыночной зарплатной вилки на основе опыта, региона и стека технологий; парсинг загруженных файлов резюме.

---

## 🛠 Технологический стек
- **Backend:** Python 3.10+, Django 5, Django REST Framework, Llama.cpp, PDFPlumber, python-docx
- **Frontend:** Node.js 18+, React, Vite, TailwindCSS, Radix UI
- **Machine Learning:** CatBoost, Pandas, Scikit-learn

## Установка и запуск (Локально)

### Шаг 1. Подготовка ML моделей и данных
Чтобы сервис работал корректно и не отдавал заглушки, вам нужно поместить рабочие файлы моделей в нужные папки:

1. **CatBoost и база вакансий:**
   Создайте папку `data` в корне проекта (если её нет) и поместите туда файлы:
   - `data/model.cbm`
   - `data/vacancies.csv`

2. **Модель для парсинга резюме (LLaMA/Qwen):**
   Поместите `.gguf` файл (например, `qwen2.5-0.5b-instruct-q4_k_m.gguf`) в папку:
   - `backend/salary/api/models/`

### Шаг 2. Запуск Backend (Django)
Убедитесь, что у вас установлен Python (рекомендуется 3.10+).

```bash
# 1. Откройте терминал в корне проекта
# 2. Создайте и активируйте виртуальное окружение
python -m venv venv
venv\Scripts\activate      # Для Windows
# source venv/bin/activate # Для Mac/Linux

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Перейдите в папку бэкенда и примените миграции
cd backend/salary
python manage.py migrate

# 5. Запустите сервер
python manage.py runserver
```
Сервер будет доступен по адресу: `http://127.0.0.1:8000`

### Шаг 3. Запуск Frontend (React)
Убедитесь, что у вас установлен Node.js (v18+).

```bash
# 1. Откройте новый терминал и перейдите в папку фронтенда
cd frontend/salary-range-hackaton-frontend

# 2. Установите зависимости
npm install

# 3. Запустите dev-сервер
npm run dev
```
Фронтенд запустится и будет доступен по адресу: `http://localhost:5173`

## Описание API
Бэкенд предоставляет несколько ключевых эндпоинтов. Вы можете посмотреть интерактивную документацию Swagger UI при запущенном сервере:
**[http://127.0.0.1:8000/api/schema/swagger-ui/](http://127.0.0.1:8000/api/schema/swagger-ui/)**

Основные методы:
- `POST /api/calculate_salary/` — принимает распарсенные данные (роль, навыки, опыт, регион) и возвращает рассчитанную вилку `min_salary`, `max_salary` и массив `recommendations`.
- `POST /api/parse_resume/` — принимает файл (PDF/DOCX/TXT), пропускает через локальную LLM и возвращает структурированный JSON с навыками и опытом.

## Обучение модели
Если вы хотите переобучить модель предсказания вилок, используйте скрипты в корне проекта (например, `train_fork_model.py`). 
Скрипт загружает сырой файл с вакансиями и генерирует на выходе обновленный `model.cbm` и базу `vacancies.csv`.
